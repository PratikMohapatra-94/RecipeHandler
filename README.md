# RecipeHandler
Spring Boot Project for Recipe Management

Project :Maven 
Language :Java 
Version :8
Packaging :JAR

Dependencies:
Swagger2
JPA
H2 
Junit
Mockito
DevTools

Consumable URL:
http://localhost:8080/swagger-ui.html#/
http://localhost:8080/h2-console/
http://localhost:8080/add
http://localhost:8080/remove/{recipeId}
http://localhost:8080/update/{recipeId}
http://localhost:8080/getAll
http://localhost:8080/getById/{recipeId}
